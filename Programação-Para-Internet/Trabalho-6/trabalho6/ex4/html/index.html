Explicação da Saída do Código JavaScript - Árvore DOM

O Código Original

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Nós na Árvore DOM</title>
  <style>
    img { width: 100px; }
  </style>
</head>
<body>
  <h2>Tipos de nós na árvore DOM</h2>
  <input type="text" id="myInput">
  <!-- comentário -->
  <pre id="output"></pre>
  <script>
    const output = document.querySelector("#output");
    for (let node of document.body.childNodes)
      output.textContent += node.constructor.prototype + '\n';
    const input = document.querySelector("#myInput");
    output.textContent += (input instanceof HTMLInputElement) + '\n';
    output.textContent += (input instanceof HTMLElement) + '\n';
    output.textContent += (input instanceof Element) + '\n';
    output.textContent += (input instanceof Node) + '\n';
    output.textContent += (input instanceof EventTarget) + '\n';
    output.textContent += document.constructor.prototype + '\n';
  </script>
</body>
</html>

Explicação da Saída

A árvore DOM (Document Object Model) é uma representação em memória do documento HTML onde cada elemento, comentário ou texto é um nó na árvore.

Parte 1: Filhos do body

O código percorre todos os filhos diretos do elemento <body> e mostra o protótipo de cada um:

  • Espaços em branco: Text.prototype
  • Elemento h2: HTMLHeadingElement.prototype
  • Elemento input: HTMLInputElement.prototype
  • Comentário: Comment.prototype
  • Elemento pre: HTMLPreElement.prototype
  • Elemento script: HTMLScriptElement.prototype

Parte 2: Herança do elemento input

O código verifica a hierarquia de herança do elemento input:

Verificação Resultado
input instanceof HTMLInputElement true
input instanceof HTMLElement true
input instanceof Element true
input instanceof Node true
input instanceof EventTarget true

Parte 3: Protótipo do documento

A última linha mostra o protótipo do objeto document: HTMLDocument.prototype
